# Wefee

主页：[Wefee.cc](http://wefee.cc)  
文档首页：[wefee.io](http://wefee.io)
QQ群：487598183

`wefee` 是基于 `ThinkPHP5` 开发的微信管理系统。她设计的初衷是方便开发人员快速的基于 `ThinkPHP5` 进行微信Application开发，她只专注于单一公众号的开发与管理，并不是市场上其他的多公众号微信管理系统。
